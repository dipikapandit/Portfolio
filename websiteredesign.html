Website Redesign

Homepage and UI redesign focused on clarity, hierarchy, and modern usability.

Role: UI / Web Designer
Type: Website Redesign
Year: 2025

Project Overview

This project focused on redesigning an existing website to improve visual hierarchy, readability, and overall user experience. The original site contained dense content and inconsistent layout patterns, making it difficult for users to navigate and understand key information.

Problems Identified

Key issues included weak visual hierarchy, lack of spacing, outdated typography, and unclear calls to action. Important content was buried, leading to poor scannability and user fatigue.

Original website — visual clutter and poor hierarchy

Design Approach

The redesign prioritized clear content structure, consistent spacing, and a modern typographic system. Sections were reorganized to guide users naturally through the page, with emphasis placed on key actions.

Layout restructuring and hierarchy exploration

Final Design

The final design presents a cleaner interface with improved readability, modern UI elements, and a more professional visual tone. The layout adapts smoothly across devices while maintaining consistency.

Outcome & Learnings

This project strengthened my ability to analyze existing interfaces, identify usability issues, and translate them into thoughtful design solutions. It reinforced the importance of hierarchy, spacing, and content-first design decisions.
